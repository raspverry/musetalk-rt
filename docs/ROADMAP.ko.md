# 로드맵

- 언어: [EN](ROADMAP.md) | **KO** | [JA](ROADMAP.ja.md)

## 현재 단계: 검증 스택 -> 카나리 준비 런타임 후보

### 트랙 A: 휴먼 근거 완성
- QA 팩 기반 리뷰 세션 수행
- stable/bursty/jittery 스코어카드 작성
- 의사결정 결과 추세 확인

### 트랙 B: 카나리 증거 패키지
- 신뢰성/스트레스/휴먼 점수 증거 결합
- 롤백 트리거/책임자 명확화
- GO/WARN/NO-GO 기준으로 게이팅

### 트랙 C: 제품 준비성 전 단계 근거
- 평가자 수/세션 수 확대
- 모바일/네트워크 유사 조건에서 지각 품질 점검
- broader readiness 기준 확정

## 현재 범위 밖
- 프로덕션 오케스트레이션 확장
- 최종 제품 UI 범위
- 대규모 배포 약속
