# Roadmap

- Language: **EN** | [KO](ROADMAP.ko.md) | [JA](ROADMAP.ja.md)

## Phase now: validation stack -> canary-ready runtime candidate

### Track A: Human evidence completion
- Execute reviewer sessions using QA pack.
- Fill scorecards for stable/bursty/jittery profiles.
- Evaluate decision summaries and trend over time.

### Track B: Canary evidence package
- Bundle reliability + stress + human scoring evidence.
- Confirm rollback triggers and owner responsibilities.
- Gate on decision outcomes (`GO_WARN_ONLY_CANARY`, etc.).

### Track C: Pre-product-readiness evidence
- Expand reviewer coverage and inter-rater agreement checks.
- Add perceptual checks under realistic network/mobile-like conditions.
- Define clear criteria for broader readiness claim.

## Out of scope right now
- Production orchestration rollout
- Final product UI scope
- Broad deployment commitments
