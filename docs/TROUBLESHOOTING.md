# Troubleshooting

- Language: **EN** | [KO](TROUBLESHOOTING.ko.md) | [JA](TROUBLESHOOTING.ja.md)

## Common issues
- Missing report files: run scenario/harness again.
- `NO_GO_INSUFFICIENT_HUMAN_QA`: fill scorecard values.
- Streamlit not found: `pip install streamlit`.
- Empty VRAM metrics: `nvidia-smi` unavailable in environment.

## Scope reminder
If results look good, that still does not mean production readiness; keep canary flagged and non-production.
